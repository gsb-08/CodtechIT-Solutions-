# CodTech IT Solutions — Data Analytics Internship

This repository contains the deliverables, Power BI reports (`.pbix`), and documentation completed during my Data Analytics Virtual Internship at **CodTech IT Solutions**.

---

## 📑 Overview & Objectives

During this internship, I developed interactive Power BI dashboards to transform raw sales, performance, and operational data into actionable business insights.

Key responsibilities included:
- Data cleaning and transformation using Power Query.
- Data modeling and establishing relationships between datasets.
- Writing DAX measures to calculate key performance indicators (KPIs).
- Designing user-friendly visual layouts and interactive dashboards.

### 1. Hyperstore Sales Analysis
- **File:** `reports/Hyperstore_Sales_Analysis.pbix`
- **Focus:** Exploratory data analysis of retail sales across product categories, regions, and customer segments.
- **Key Insights:** Identified top-performing revenue regions and seasonal trends.

### 2. Analytics Task 1 & Task 2
- **Files:** `reports/Task_1_Sales_Dashboard.pbix`, `reports/Task_2_Analysis.pbix`
